Documentação da API Node.js - Users, Tasks e Categories

Esta API expõe endpoints REST para gerenciar usuários, tarefas e categorias. Todos aceitam e retornam JSON.

1. Users

  • GET /users – lista todos os usuários.
  • POST /users – cria um usuário.
    Exemplo de corpo:
    { "name": "Ana", "email": "ana@exemplo.com", "password": "123456" }
  • GET /users/:id – detalhes de um usuário pelo ID.
  • PUT /users/:id – atualiza nome e email do usuário.
  • DELETE /users/:id – remove usuário.
  • POST /login – login do usuário.
    Exemplo de corpo:
    { "email": "ana@exemplo.com", "password": "123456" }

2. Tasks

  • GET /task – lista todas as tarefas.
  • POST /task – cria uma tarefa.
    Exemplo de corpo:
    { "user_id": 1, "title": "Estudar Node.js", "description": "Estudar os módulos e rotas", "date_creation": "2025-05-23", "date_delivery": "2025-05-30", "status": "pendente" }
  • PUT /task/:id – atualiza título, descrição e status da tarefa.
  • DELETE /task/:id – exclui tarefa.

3. Categories

  • GET /category – lista todas as categorias.
  • POST /category – cria uma categoria.
    Exemplo de corpo:
    { "name": "Trabalho", "description": "Tarefas relacionadas ao trabalho" }
  • GET /category/:id – detalhes da categoria pelo ID.
  • PUT /category/:id – atualiza nome e descrição da categoria.
  • DELETE /category/:id – exclui categoria.

Exemplos de uso via curl

curl -X POST http://localhost:3000/users \
     -H "Content-Type: application/json" \
     -d '{"name":"Ana","email":"ana@exemplo.com","password":"123456"}'

curl http://localhost:3000/task

curl -X POST http://localhost:3000/category \
     -H "Content-Type: application/json" \
     -d '{"name":"Trabalho","description":"Tarefas relacionadas ao trabalho"}'
